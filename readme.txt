Fly från fängelset

Beskrivning

"Fly från fängelset" är ett textbaserat äventyrsspel där spelaren måste använda sin list för att fly från en fängelsecell. Genom att göra olika val påverkas spelets gång, och det finns flera möjliga sätt att rymma.

Spelregler

Läs beskrivningen av situationen noggrant.

Gör dina val genom att ange rätt siffra och trycka på "Enter".

Om du misslyckas kan du starta om spelet och försöka igen.


Funktioner

Flera valmöjligheter som påverkar spelets utgång.

Olika sätt att fly, inklusive att smyga, lura vakterna eller använda föremål.

Flera slut beroende på spelarens val.
